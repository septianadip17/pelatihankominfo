# Program Nasi Kotak
## _Restoran Terserah_
Selamat datang di Program Nasi kotak!
Program ini adalah sebuah program sederhana yang dirancang untuk membantu pengguna memesan nasi kotak secara online. 

## Fitur
- Menampung nama, jumlah pesanan dan nomor telepon user.
- Tersedia harga awal pemesanan.
- Cabang yang sudah mulai merambah ke daerah kota.

## Teknologi
- PHP
- CSS
- HTML
- JSON
